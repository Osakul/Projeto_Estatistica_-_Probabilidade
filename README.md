Bem-vindo ao repositório do projeto da disciplina de **Estatística e Probabilidade**. 

Este projeto utiliza uma base de dados reais da indústria de videogames (Kaggle) para prever se um jogo tem potencial para se tornar um "Hit Comercial" (ultrapassar a marca de 1 milhão de cópias vendidas globalmente). Para isso, construímos análises probabilísticas e treinamos modelos de inteligência artificial.

## ⚙️ Como executar o projeto localmente (Para Professores/Avaliadores)

Caso deseje rodar o ambiente na sua própria máquina, siga o passo a passo abaixo. 

1. Faça o clone deste repositório:

  git clone [https://github.com/Osakul/Projeto_Estatistica_-_Probabilidade](https://github.com/Osakul/Projeto_Estatistica_-_Probabilidade.git)

2. Acesse a pasta do projeto:

  cd [NOME_DO_SEU_REPOSITORIO]

3. Caso você seja um usuário de Linux, crie e ative um ambiente virtual isolado:

  python3 -m venv venv <br>
  source venv/bin/activate

4. Instale as dependências contidas no arquivo requirements.txt:

  pip install -r requirements.txt

5. Inicie o servidor do dashboard web:

  python3 -m streamlit run app.py
